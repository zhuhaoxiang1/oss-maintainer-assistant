# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language.
- Being respectful of differing viewpoints and experiences.
- Gracefully accepting constructive criticism.
- Focusing on what is best for the community.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior. Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this code.
