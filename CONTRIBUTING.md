# Contributing

Thanks for helping improve `oss-maintainer-assistant`.

## Development

```sh
npm install
npm test
```

## Pull Requests

- Keep changes small and focused.
- Add or update tests for behavior changes.
- Do not commit secrets, `.env` files, tokens, or generated credentials.
- Keep GitHub automation dry-run first unless a maintainer explicitly approves a mutation feature.

## Issues

Good bug reports include:

- The command you ran.
- Expected behavior.
- Actual behavior.
- Node.js version.
- Whether the issue happens with `--dry-run`.
